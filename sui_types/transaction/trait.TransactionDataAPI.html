fn process_funds_withdrawals( &self, ) -> UserInputResult<BTreeMap<AccumulatorObjId, u64>>

Processes funds withdraws and returns a map from funds account object ID to total reserved amount. This method aggregates all withdraw operations for the same account by merging their reservations. Each account object ID is derived from the type parameter of each withdraw operation.
